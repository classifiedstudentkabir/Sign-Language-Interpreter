**⭐ Slide 5 — How SignLens Works (Simple Workflow)**

1. User signs → webcam captures
2. MediaPipe detects hand landmarks
3. TensorFlow model predicts the gesture
4. Text appears on screen
5. Web Speech API converts text → speech
6. Teacher’s voice → speech-to-text → student sees text
