# Gesture Guide

This guide explains how to perform the hand gestures recognized by the application.

## Single-Hand Gestures

| Gesture | How to Perform |
| --- | --- |
| **Number 0** | ✊ Make a fist with all fingers curled, including the thumb. |
| **Number 1** | ☝️ Extend only the index finger upwards. Curl all other fingers. |
| **Number 2** | ✌️ Extend the index and middle fingers (V-shape). The thumb must be curled IN. |
| **Number 3** | 🖐️ Extend the index, middle, and ring fingers. The thumb must be curled IN. |
| **Number 4** | 🖐️ Extend all four fingers (index, middle, ring, pinky). The thumb must be curled IN. |
| **Number 5** | 🖐️ Extend all five fingers, including the thumb. |
| **Thumbs Up** | 👍 Make a fist and point your thumb upwards. |
| **Thumbs Down**| 👎 Make a fist and point your thumb downwards. |
| **Okay** | 👌 Touch your thumb and index finger to form a circle. Extend the other three fingers. |
| **Thank You** | ✌️ Extend the index and middle fingers (V-shape). The thumb must be extended OUT. |
| **Call Me** | 🤙 Extend the thumb and pinky finger. Curl the index, middle, and ring fingers. |

## Two-Hand Gestures

These gestures require a specific gesture on each hand simultaneously.

| Gesture | Right Hand | Left Hand | How to Perform |
| --- | --- | --- | --- |
| **YES** | 👍 Thumbs Up | 👌 Okay | Right hand gives a thumbs up, left hand makes the okay sign. |
| **NO** | 👎 Thumbs Down | 👌 Okay | Right hand gives a thumbs down, left hand makes the okay sign. |
| **HELP** | 👍 Thumbs Up | ✌️ Thank You | Right hand gives a thumbs up, left hand makes the "Thank You" sign. |
| **Good Job** | 👍 Thumbs Up | 👍 Thumbs Up | Both hands give a thumbs up. |
| **Perfect** | 👌 Okay | 👌 Okay | Both hands make the okay sign. |
| **Accepted** | 🖐️ Number 5 | 👍 Thumbs Up | Right hand is an open palm, left hand gives a thumbs up. |
| **Rejected** | 🖐️ Number 5 | 👎 Thumbs Down | Right hand is an open palm, left hand gives a thumbs down. |
| **Ready** | ✊ Number 0 (Fist) | 👍 Thumbs Up | Right hand makes a fist, left hand gives a thumbs up. |
| **Select** | ✌️ Thank You | ☝️ Number 1 | Right hand makes the "Thank You" sign, left hand points with index finger. |
| **Alert** | 🖐️ Number 5 | ✌️ Thank You | One hand is an open palm, the other makes the "Thank You" sign (works both ways).|
| **All Good** | 👌 Okay | 👍 Thumbs Up | Right hand makes the okay sign, left hand gives a thumbs up. |
| **We built SignLens.** | ✊ Number 0 | ✋ Number 5 | One hand makes a fist, the other is an open palm. |
| **Good Morning** | ✊ Number 0 | ☝️ Number 1 | One hand makes a fist, the other shows 1. |

| **Good Evening** | ✊ Number 0 | 🖐️ Number 3 | One hand makes a fist, the other shows 3. |

| **Bye** | ✌️ Thank You | ✌️ Thank You | Both hands make the "Peace" sign. |
